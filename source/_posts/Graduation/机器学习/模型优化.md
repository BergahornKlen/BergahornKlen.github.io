# 模型优化

## 数据

>   数据的质量决定模型表现的上限

检查：

1.   数据**属性的意义**，是否为无关数据 => 
     删除不必要的属性 => 减少过拟合、节约运算时间
2.   不同属性数据的**数量级**差异性如何 =>
     数据**预处理**：归一化、标准化 => 平衡数据影响，加快训练收敛
3.   是否有**异常**数据 =>
     是否保留或过滤掉**异常**数据 => 提高鲁棒性
4.   数据样本**数量**是否足够 => 扩大数据样本
5.   **采集数据**的方法是否合理，采集到的数据是否有代表性
6.   对于标签结果，要确保**标签判定规则**的一致性（统一标准）



## 不同的模型

尝试不同的模型，对比模型表现 => 帮助确定更合适的模型
